# Kimwanyi SACCO Management System

## Overview

The **Kimwanyi SACCO Management System** is a web-based application developed to digitize the daily operations of Kimwanyi SACCO, replacing the existing paper-based record-keeping system with a centralized and secure digital platform. The system improves efficiency, accuracy, and accountability in managing members, savings, loans, and financial transactions.

The application enables SACCO staff to register and manage members, process savings deposits and withdrawals, handle loan applications and repayments, calculate interest, and generate account statements. It also provides administrators with an overview of the SACCO's operations through a dashboard containing key financial summaries and statistics.

By automating these processes, the system minimizes human errors, prevents inconsistent records, improves loan tracking, and provides secure storage of member and transaction data.

## Key Features

* Member registration and profile management
* Secure user authentication and role-based access
* Savings deposits and withdrawals
* Savings balance and transaction history
* Loan application, approval, rejection, and repayment
* Automatic savings and loan interest calculations
* Member account statement generation
* Administrative dashboard with financial summaries
* Enforcement of SACCO business rules and validations

## Technologies Used

* **Programming Language:** Java (Core Java)
* **Web Framework:** JavaServer Faces (JSF)
* **ORM Framework:** Hibernate
* **Database:** MySQL
* **Build Tool:** Maven
* **Application Server:** Apache Tomcat

## Objective

The primary objective of this project is to provide Kimwanyi SACCO with a reliable and user-friendly management system that streamlines its operations, improves data integrity, enhances service delivery to members, and eliminates the challenges associated with manual record management.
